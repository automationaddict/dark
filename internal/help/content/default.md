# Help

Context-aware help for this view hasn't been written yet.

## Global shortcuts

- `?` open or close this help drawer
- `F1` – `F12` switch tabs
- `ctrl+r` rebuild and hot-reload dark
- `ctrl+c` / `q` quit

## Settings (F1)

- `j` / `k` or arrow keys move between sidebar sections
- `+` / `-` resize the sidebar

## Inside this drawer

- `j` / `k` scroll
- `[` / `]` jump to previous or next heading
- `/` start search, `n` / `N` next / prev match
- `esc` close
